# 功能

## 画质

登录后可以解锁更高画质，如何登录见 [上传篇](../bilibili-upload.md#登录B站账号)

## 弹幕

支持的弹幕类型：

- 文本弹幕
- 普通礼物弹幕
- 舰长礼物弹幕
- SC礼物弹幕

### 弹幕无法录制

1. 检查录制是否配置账户
2. 网页登录B站直播间，检查是否出现验证码并进行处理

## 支持的链接

`https://live.bilibili.com/房间号`

# 常见问题

## 412风控如何处理

1. 调整使用“批量查询接口”参数
2. 调大检查间隔
3. 修改“并发数”为1
4. 调大“等待时间”为1000ms
